BMI Calculator

Description
A simple Python program that calculates the Body Mass Index (BMI) based on the user's weight and height. It then categorizes the BMI into Underweight, Normal, Overweight, or Obese.

This project helps beginners practice:
- Taking user input
- Performing calculations
- Conditional statements (if/elif/else)
- Printing results



How to Run
1. Open `main.py` in a Python environment.
2. Run the program.
3. Enter your weight (in kilograms) and height (in meters) when prompted.
4. The program will display your BMI and the category.
