Voice Assistant (Basic)

Description
A simple Python-based voice assistant that can respond to basic voice commands. It can:
- Respond to greetings
- Tell the current time
- Tell the current date
- Perform a Google search based on user input

This project helps beginners practice:
- Using Python libraries
- Conditional statements
- Basic automation
- Voice input/output handling

How to Run
1. Install required libraries
2. Open main.py in a Python environment.
3. Run the program.
4. Speak or type commands as prompted.
5. The assistant will respond or perform actions accordingly.
