OIBSIP - Oasis Infobyte Python Internship Projects

This repository contains all the projects completed as part of the Oasis Infobyte Python Internship (January 2026 batch).

The goal of this internship was to strengthen Python programming skills and problem-solving abilities through hands-on projects. This repository includes 3 projects, each demonstrating practical Python applications.

---

Projects Included

1. BMI Calculator - Calculates Body Mass Index and categorizes the result.
2. Random Password Generator - Generates secure random passwords based on user input.
3. Voice Assistant (Basic) - A simple Python-based voice assistant that can respond to basic commands.
