Random Password Generator

Description
A simple Python program that generates random passwords based on user input. Users can specify the length of the password, and the program generates a secure password containing letters, numbers, and symbols.

This project helps beginners practice:
- Loops
- Strings
- Randomization
- User input handling

How to Run
1. Open `main.py` in a Python environment.
2. Run the program.
3. Enter the desired password length when prompted.
4. The program will generate and display a random password.
